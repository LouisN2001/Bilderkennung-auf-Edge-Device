# Security Policy

This repository contains the code for the two completed academic student research projects.

### Project Status & Support
This project is an **academic archive**. It is **not actively maintained**, monitored, or updated. No security patches or updates will be provided for the application, its dependencies, or the integrated frameworks.

### Reporting a Vulnerability
If you notice any security vulnerabilities in this project, please note that they will not be fixed in this repository. 
* If you or your university plan to continue or deploy this project, it is highly recommended to review and update all dependencies listed in the `requirements.txt` file before running the application in a production environment.
* The software is provided "as is", without warranty of any kind.
